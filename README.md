# Winnie Logo 项目

## 项目说明

这是一个带有动画效果的 Winnie Logo 网页。

## 功能特点

### 1. 小猫走路动画
- 使用 MP4 视频文件 (`制作猫咪走路视频.mp4`) 展示小猫走路
- 小猫从右向左行走
- 走路完成后自动切换为静态图片

### 2. 静态小猫定位
- 小猫走路完成后停在字母 **e** 的右下方（`left: 540px`, `bottom: -15px`）
- 使用 PNG 图片 (`微信图片_20260207230441_24501_12.png`) 作为静态坐下姿势

### 3. 文字动画
- "Winnie" 的每个字母依次落下动画
- 带有弹跳和发光效果
- 字母颜色各异，带有阴影效果

### 4. 装饰元素
- 维尼熊耳朵装饰
- 小米风格边框
- 闪光粒子效果
- 背景渐变图案

### 5. 爱心装饰
- 在 subtitle 中添加了爱心符号：WINNIE ❤️ 维尼魅力

## 动画时间线

| 时间 | 动画内容 |
|------|----------|
| 0s | 字母 W 开始落下 |
| 0.2s | 字母 i 开始落下 |
| 0.4s | 字母 n 开始落下 |
| 0.6s | 字母 n 开始落下 |
| 0.8s | 字母 i 开始落下 |
| 1.0s | 字母 e 开始落下 |
| 2.0s | 边框动画 |
| 2.5s | 副标题淡入 |
| 5.2s | 小猫走路完成，切换为静态图片 |

## 交互功能

- **点击 logo**：重新播放所有动画

## 技术栈

- HTML5
- CSS3 (动画、渐变、阴影)
- JavaScript (动画控制)
- Video 元素 (MP4 播放)
